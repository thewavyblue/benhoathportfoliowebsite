My Wedding R.S.V.P Website

My Wedding R.S.V.P Website

I designed and built this website in response to the many wedding R.S.V.P websites offering templates, calendars, and all the other functionalities for a subscription fee. As I’m currently learning web development it seemed only natural to undertake this as my first real tangible, functional, and useful project to flex the knowledge I’ve gained so far.

This project took me around the houses a bit, with the most challenging part being the ‘wedding gift’ process – asking for fiscal contributions in as elegant a way as possible is tricky, to say the least. What was meant to be a simple static microsite turned into a WordPress-driven website featuring a donation plugin and some PHP script… and PHP hadn't even been featured in my learning material at the time! Still, I wasn't going to let that deter me... what I didn't know already and went and learned, iterated, and implemented.

See this website live at benandjentietheknot.site
